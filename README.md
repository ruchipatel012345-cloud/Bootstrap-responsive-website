 # Responsive Restaurant & Bar Website

A fully responsive restaurant and bar website built using HTML, CSS, and Bootstrap 5.

## Features
- Responsive design for all devices
- Menu section
- Gallery section
- Contact form
- Reservation section

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
